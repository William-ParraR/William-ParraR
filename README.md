## 🔭 Experiencia destacada

### 📡 Tesis — Banco de Pruebas para Correladores FX y XF
> *Desarrollo de un banco de pruebas para un sistema de correlación enfocado a un sistema interferométrico de dos antenas* · Universidad Industrial de Santander
- Diseño y desarrollo de un banco de pruebas para validar correladores FX y XF orientados a radioastronomía interferométrica (línea de hidrógeno neutro, 21 cm)
- Caracterización de dispositivos RF, cables y componentes de radiofrecuencia
- Manejo de instrumentación: VNA, analizador de espectro, generador de señales, osciloscopio

### 🛫 Proyecto UIS × ANE — Monitoreo de Frecuencias Aeronáuticas
> Colaboración entre la Universidad Industrial de Santander y la Agencia Nacional del Espectro
- Implementación de una estación de monitoreo para frecuencias de comunicación entre aviones y torres de control
- Caracterización del espectro radioeléctrico en **3 ciudades de Colombia**
- Análisis de interferencias y calidad de señal en banda VHF aeronáutica
